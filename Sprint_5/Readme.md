# EX5: Conditional Formatting

---

You are working as a Junior Data Analyst in the Business Intelligence Team at DMart.
The management team reviews daily sales reports containing thousands of product records.
```
However, identifying:
- Top-performing products
- Low-performing products
- Revenue trends
- Loss-making items
- High discount products

is difficult by looking only at raw numbers.
The manager Rajesh has asked you to use Conditional Formatting in Excel to visually highlight important business insights.
```

---

# Conditional Formatting Rules to Implement

```

| Rule Type            | Business Purpose         |
| -------------------- | ------------------------ |
| Highlight Cell Rules | Identify high/low values |
| Top/Bottom Rules     | Top 10 products          |
| Data Bars            | Revenue comparison       |
| Color Scales         | Profit trend analysis    |
| Icon Sets            | KPI indicators           |
| Duplicate Values     | Detect repeated records  |
| Formula-Based Rules  | Custom business logic    |
```
